# Archive

This directory preserves material from earlier research iterations that is not
part of the maintained 2D heat-equation PINO implementation.

- `future_work/`: fractional-PINO, HPFRACC, JAX/CuPy, and biophysics extensions.
- `research_notes/`: previous paper drafts, enhancement notes, and roadmap docs.
- `results/`: prior generated result summaries.
- `papers/`: reference PDF from the original project.

Archived code is kept for reference only. It is not imported by `src`, covered by
the active tests, or required by `requirements.txt`.
